  • Sea \(\lambda_0>0\). Cul es la prueba Wald para \(H_0: \lambda = \lambda_0, H_1: \lambda \neq \lambda_0\)
wald_poisson <- function(x, lambda_0){
  lambda_1 <- mean(x)
  se_lambda <- sqrt(lambda_1 / length(x))
  z <- (lambda_1 - lambda_0) / se_lambda
  p_value <- 2 * pnorm(-abs(z))
  p_value
}

2. Relacin entre bootstrap e inferencia bayesiana

Consideremos el caso en que tenemos una nica observacin \(x\) proveniente de una distribucin normal

\[x \sim N(\theta, 1)\]

Supongamos ahora que elegimos una distribucin inicial Normal.

\[\theta \sim N(0, \tau)\]

dando lugar a la distribucin posterior (como vimos en la tarea)

\[\theta|x \sim N\bigg(\frac{x}{1 + 1/\tau}, \frac{1}{1+1/\tau}\bigg)\]

Ahora, entre mayor \(\tau\), ms se concentra la posterior en el estimador de mxima verosimilitud \(\hat{\theta}=x\). En el lmite, cuando \(\tau \to \infty\) obtenemos una inicial no-informativa (constante) y la distribucin posterior

\[\theta|x \sim N(x,1)\]

Esta posterior coincide con la distribucin de bootstrap paramtrico en que generamos valores \(x^*\) de \(N(x,1)\), donde \(x\) es el estimador de mxima verosimilitud.

Lo anterior se cumple debido a que utilizamos un ejemplo Normal pero tambin se cumple aproximadamente en otros casos, lo que conlleva a una correspondencia entre el bootstrap paramtrico y la inferencia bayesiana. En este caso, la distribucin bootstrap representa (aproximadamente) una distribucin posterior no-informartiva del parmetro de inters. Mediante la perturbacin en los datos el bootstrap aproxima el efecto bayesiano de perturbar los parmetros con la ventaja de ser ms simple de implementar (en muchos casos).
*Los detalles se pueden leer en The Elements of Statistical Learning de Hastie y Tibshirani.

dinv_gamma <- function(sigma2, alpha, beta){
  # WIKIPEDIA: {\displaystyle f(x;\alpha ,\beta )={\frac {\beta ^{\alpha }}{\Gamma (\alpha )}}(1/x)^{\alpha +1}\exp \left(-\beta /x\right)}

simulacion_posterior <- function(x, alpha, beta, n){
  alpha_posterior <- alpha + length(x) / 2
  beta_posterior <- beta + sum(x^2) / 2
  posterior <- 1 / rgamma(n=n, shape=alpha_posterior, rate=beta_posterior)
  posterior
